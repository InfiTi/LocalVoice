# LocalVoice 开发日志

## 2026-07-14 阶段 1 MVP 开发

### 完成内容
1. **项目目录结构搭建**
   - `config/settings.ini` — 全局配置文件
   - `core/` — 核心模块（分句、预处理、TTS引擎、播放器、队列）
   - `hotkey/` — 快捷键模块
   - `models/` — 模型存放目录
   - `temp/` — 临时音频缓存

2. **智能分句模块 (`sentence_splitter.py`)**
   - 支持中英文标点分句（。！？；!?;）
   - 处理引号内嵌句号（不在引号内断句）
   - 单句长度上限控制（默认 200 字，超长自动切分）
   - 最小朗读单元控制（太短不断句）

3. **文本预处理模块 (`text_preprocess.py`)**
   - 去除换行冗余、多余空白
   - 过滤特殊符号、控制字符
   - 文本去重（防止短时间重复复制）
   - 与分句模块联动

4. **TTS 引擎抽象层 (`tts_engine.py`)**
   - `BaseTTSEngine` 抽象基类，定义统一接口
   - `PiperTTSEngine` 适配器，对接 Piper ONNX 模型
   - 支持 onnxruntime 推理，输出 WAV 音频
   - 语速、音量参数透传

5. **音频播放控制 (`audio_player.py`)**
   - 基于 pygame.mixer，支持暂停/继续/停止
   - 流式播放：逐句合成 → 逐句播放（边播边合成）
   - 音量实时调节
   - 播放状态回调通知

6. **朗读队列管理 (`tts_queue.py`)**
   - 线程安全的朗读队列
   - 流式处理：先切句 → 首句立即合成播放 → 后续句子异步排队
   - 新文本插入时自动停止当前播放并清空旧队列
   - 队列状态查询

7. **快捷键触发模块 (`hotkey_control.py`)**
   - 基于 keyboard 库
   - Ctrl+Alt+R：选中文本触发朗读
   - Ctrl+Alt+P：暂停/继续
   - Ctrl+Alt+X：停止
   - Ctrl+Alt+S：开关自动监听模式（阶段 2）
   - Ctrl+Alt+↑/↓：加速/减速（阶段 3）

8. **主程序入口 (`main.py`)**
   - 串联所有模块
   - 初始化配置、引擎、播放器、队列
   - 注册快捷键
   - 后台常驻运行

### 技术决策记录
1. **音频播放选 pygame.mixer 而非 playsound** — playsound 不支持暂停/继续/调速，pygame.mixer 完整支持
2. **流式分句策略** — 复制大段文字后，先切句，首句立即合成播放，后续句子在后台线程异步合成并排队，保证"快速听到第一句"
3. **keyboard 库而非 global-hotkey** — keyboard 同时支持热键和键监听，减少依赖
4. **分句单独成模块** — 分句质量直接决定朗读体验，后续可独立升级（规则→模型）
