# LocalVoice 开发路线图

> 本地离线轻量化 TTS 文本朗读工具，支持全局文本抓取、流式分句播放、模型热插拔。

## 项目目标

解决"长时间阅读文字导致眼睛疲劳"的问题，通过本地离线 TTS 将文字转为语音，实现：
- 任意场景复制/选中文本 → 自动或一键触发朗读
- 流式分句处理，复制大段文字也能快速听到第一句
- 全局快捷键控制（暂停、停止、调速、开关）
- 模型热插拔，切换不同 TTS 引擎无需改业务代码

## 技术栈

| 组件 | 选型 | 说明 |
|------|------|------|
| 核心语言 | Python 3.10+ | 兼容所有轻量 TTS 模型 |
| TTS 推理 | onnxruntime | 通用模型推理框架 |
| 音频播放 | pygame | 支持暂停/继续/音量/流式播放 |
| 快捷键 | keyboard | 全局热键 + 键监听 |
| 剪贴板 | pyperclip | 轻量跨平台 |
| 配置 | configparser | Python 内置 |
